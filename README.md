# 🎓 Internship Management Platform

> **University of Jendouba** · Final Year Thesis (PFE) · Score: **18/20 — "Très Bien"** · 2023/2024

A full-stack web platform designed to manage mandatory internships for students at the University of Jendouba — digitizing the entire process from request submission to final approval.

---

## 📌 Overview

This platform connects three types of users in a streamlined internship management workflow:

| Role | Capabilities |
|------|-------------|
| 🎓 **Student** | Submit internship requests, upload documents, track status |
| 👨‍🏫 **Supervisor** | Review requests, validate documents, approve/reject |
| 🛠️ **Admin** | Oversee all internships, manage users, generate reports |

---

## 🚀 Features

- ✅ Role-based access control (Student / Supervisor / Admin)
- ✅ Internship request submission and tracking
- ✅ **QR-code based approvals** — scan to validate documents instantly
- ✅ Document uploads (internship reports, contracts)
- ✅ Automatic status tracking and notifications
- ✅ Admin dashboard with full oversight
- ✅ Responsive Angular frontend

## ▶️ How to Run

**Backend:**
```bash
cd backend
npm install
node server.js
```

**Frontend:**
```bash
cd front_end
npm install
ng serve
```

Then open 👉 `http://localhost:4200`
